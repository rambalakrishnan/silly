Schema 1: fragrance_dossier_entry
This schema defines the structure for a single, comprehensive entry about a specific fragrance within our knowledge base.


{
  "type": "object",
  "description": "A comprehensive dossier entry for a single fragrance, detailing its metadata, user acquisition history, and a detailed analysis of the user's reaction to it.",
  "properties": {
    "fragrance_name": {
      "type": "string",
      "description": "The full, official name of the fragrance, including the house and any specific concentration or flanker name (e.g., 'Elysian Fraiche (Roja Dove - Elysian Parfum)')."
    },
    "house": {
      "type": "string",
      "description": "The name of the primary fragrance house that released the original scent."
    },
    "perfumer": {
      "type": "string",
      "description": "The name of the credited perfumer or perfumers. Use 'Unidentified' if unknown."
    },
    "manufacturer": {
      "type": "string",
      "description": "The parent company or primary manufacturer/fragrance house that produces the scent (e.g., 'LVMH', 'Givaudan', 'Puig')."
    },
    "montagne_equivalent": {
      "type": "string",
      "description": "The name of the Montagne Parfums interpretation of this fragrance. Use 'N/A' if it is an original not cloned by Montagne or a Montagne Original."
    },
    "acquisition_history": {
      "type": "array",
      "description": "A chronological log of how and when the user acquired this fragrance, including format (full bottle, decant, sample) and source (Montagne, Ayybraham, secondhand, gift).",
      "items": {
        "type": "string"
      }
    },
    "emotional_resonance": {
      "type": "string",
      "description": "A qualitative analysis of the user's emotional connection to the fragrance. Describes whether it is a 'Signature' love, a 'Library' scent for personal enjoyment, a 'grower,' or elicits a specific memory or feeling."
    },
    "olfactory_analysis": {
      "type": "string",
      "description": "A technical deconstruction of the fragrance's scent profile and architecture (e.g., 'High-Contrast', 'Harmonious-Fusion'). Details how its composition aligns with or challenges the user's established preferences and the key diagnostic insights gained from testing it."
    },
    "key_notes_and_user_reaction": {
      "type": "array",
      "description": "An array of objects detailing the user's specific reaction to the most prominent notes or accords within the fragrance.",
      "items": {
        "type": "object",
        "properties": {
          "note": {
            "type": "string",
            "description": "The name of the specific olfactory note or accord (e.g., 'Grapefruit', 'Dry Tobacco Leaf')."
          },
          "reaction": {
            "type": "string",
            "description": "A qualitative description of the user's reaction to this specific note, categorized as 'Positive', 'Negative', 'Mixed', 'Neutral', 'Untested', or 'Hypothesis'."
          }
        },
        "required": ["note", "reaction"]
      }
    },
    "status_and_verdict": {
      "type": "string",
      "description": "The final, actionable conclusion for this fragrance. Classifies it as a 'Confirmed Signature Love', 'Confirmed Library Scent', 'Confirmed No', 'Successful Diagnostic', or 'Pending Test'."
    },
    "user_quotes": {
      "type": "array",
      "description": "An array containing direct, verbatim quotes from the user that are particularly insightful or representative of his experience with the fragrance.",
      "items": {
        "type": "string"
      }
    }
  },
  "required": [
    "fragrance_name",
    "house",
    "perfumer",
    "manufacturer",
    "acquisition_history",
    "emotional_resonance",
    "olfactory_analysis",
    "key_notes_and_user_reaction",
    "status_and_verdict"
  ]
}

### **Schema 2: `accord_dossier_entry`**

This schema defines the structure for a single, comprehensive entry about the user's relationship with a specific olfactory accord.

json
{
  "type": "object",
  "description": "A comprehensive dossier entry for a single olfactory accord, synthesizing the user's reactions to it across all tested fragrances to build a predictive profile.",
  "properties": {
    "accord_name": {
      "type": "string",
      "description": "The name of the specific olfactory accord or note family (e.g., 'Vanilla', 'Leather & Suede')."
    },
    "user_affinity": {
      "type": "string",
      "description": "A summary classification of the user's overall relationship with this accord, such as 'Confirmed Love (Pillar)', 'Mixed / Context-Dependent', or 'Definitive Negative'."
    },
    "olfactory_profile": {
      "type": "string",
      "description": "A brief, objective description of what the accord smells like and its role in perfumery."
    },
    "user_history_and_analysis": {
      "type": "string",
      "description": "A verbose, narrative synthesis of the user's entire history with this accord. This includes formative memories, key breakthrough tests, and an analysis of the specific contexts in which he likes or dislikes the note."
    },
    "positive_benchmarks": {
      "type": "array",
      "description": "A list of specific fragrances where the user had a confirmed positive reaction to this accord.",
      "items": {
        "type": "string"
      }
    },
    "negative_benchmarks": {
      "type": "array",
      "description": "A list of specific fragrances where the user had a confirmed negative reaction to this accord.",
      "items": {
        "type": "string"
      }
    },
    "strategic_recommendation": {
      "type": "string",
      "description": "An actionable, forward-looking strategy for this accord. Specifies whether to 'PURSUE', 'EXPLORE', 'PROCEED WITH CAUTION', or 'AVOID' this category, and lists the key pending diagnostics needed to gather more data."
    }
  },
  "required": [
    "accord_name",
    "user_affinity",
    "olfactory_profile",
    "user_history_and_analysis",
    "strategic_recommendation"
  ]
}
